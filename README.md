### Requirements

- Node

### Installation

If you have any problems please check the [Meteor docs](https://docs.meteor.com) first.

1 - Clone the project

```bash
$ git clone git@github.com:LucasLeandro1204/meteor-chat.git chat
$ cd chat
```

2 - Run meteor

```bash
$ meteor
```

Now you can go to tasklist.test/ and BOOM working =)
